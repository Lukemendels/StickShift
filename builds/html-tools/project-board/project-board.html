This is a StickShift tool. It works on its own right now - load a board, arrange cards, and copy the write envelope. StickShift just gives it a memory: it stores your boards as files and opens this tool for you.

StickShift is an Excel cockpit that gives a chat assistant a file-based memory and lets it open local tools like this one. You can use this board standalone now; come back to connect it when you're ready.
